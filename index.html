Actividad de conexión: ¿Dónde se acumula valor?

Antes de hablar de fórmulas, cada equipo analiza una situación y decide qué tipo de integral sería más pertinente.

Situaciones rápidas

  1. Una cámara virtual sigue una trayectoria curva y se desea medir el trabajo realizado por una fuerza de estabilización.
  2. Una empresa desea calcular el material necesario para cubrir una superficie irregular.
  3. Un sistema de almacenamiento desea estimar la densidad total de calor dentro de un volumen cerrado.
  4. Un equipo de analítica quiere sumar una variable de riesgo distribuida sobre una zona de operación.

Pregunta detonante

¿La acumulación ocurre sobre una trayectoria, sobre una región plana o dentro de un volumen?

Mapa conceptual de decisión

1 dimensión

Integral en línea

Se integra sobre una curva parametrizada. Sirve para longitud, masa de un cable, trabajo de un campo de fuerzas o circulación.

∫C f(x,y,z) ds    o    ∫C F · dr
2 dimensiones

Integral doble

Se integra sobre una región del plano. Sirve para área, volumen bajo una superficie, masa de una lámina o acumulación distribuida.

∫∫R f(x,y) dA
3 dimensiones

Integral triple

Se integra dentro de una región sólida. Sirve para volumen, masa, carga, calor o concentración en un cuerpo tridimensional.

∫∫∫E f(x,y,z) dV

Integrales en línea

Una integral en línea acumula una magnitud a lo largo de una curva. La curva puede representar una ruta de desplazamiento, un cable, una trayectoria de cámara, el recorrido de un robot o el flujo que sigue una partícula.

Integral de una función escalar

∫C f(x,y,z) ds = ∫ab f(r(t)) ||r'(t)|| dt

Se usa cuando la magnitud depende de la posición sobre la curva: densidad, temperatura, costo por metro o intensidad.

Integral de un campo vectorial

∫C F · dr = ∫ab F(r(t)) · r'(t) dt

Se usa para trabajo, circulación o efecto acumulado de una fuerza sobre una trayectoria.

  1. Identificar la curva C y parametrizarla como r(t).
  2. Definir el intervalo de recorrido a ≤ t ≤ b.
  3. Calcular r'(t).
  4. Si es integral escalar, calcular ||r'(t)||. Si es integral vectorial, calcular el producto punto F(r(t)) · r'(t).
  5. Integrar respecto a t.
  6. Interpretar el resultado en el contexto del problema.

Ejemplo académico

Calcular ∫C (x + y) ds, donde C está dada por r(t) = (t, t²), 0 ≤ t ≤ 1.

Desarrollo paso a paso

1. x = t, y = t², por tanto f(r(t)) = t + t².

2. r'(t) = (1, 2t), entonces ||r'(t)|| = √(1 + 4t²).

3. La integral queda:

∫01 (t + t²)√(1 + 4t²) dt

4. Esta integral puede aproximarse numéricamente si no se busca una forma cerrada simple.

Aplicación empresarial

En Ingeniería Multimedia, una trayectoria de cámara en una animación 3D puede representarse mediante una curva. Si el costo computacional por unidad de recorrido cambia según la posición de la cámara, una integral en línea permite estimar el costo total de renderizado sobre la ruta.

En Ingeniería Industrial, si un robot de inspección recorre una línea de producción y la resistencia al movimiento cambia por zonas, la integral de campo vectorial permite estimar el trabajo total requerido por el sistema.

Integrales dobles

Una integral doble acumula una cantidad sobre una región plana R. Cuando la función f(x,y) representa altura, la integral doble calcula el volumen bajo la superficie sobre R.

∫∫R f(x,y) dA

Si f(x,y) = 1, la integral calcula el área de la región R.

Región tipo I

a ≤ x ≤ b,   g1(x) ≤ y ≤ g2(x)

Se integra primero en y y luego en x.

Región tipo II

c ≤ y ≤ d,   h1(y) ≤ x ≤ h2(y)

Se integra primero en x y luego en y.

Ejemplo académico

Calcular ∫∫R (x + 2y) dA, donde R = [0,2] × [0,1].

Desarrollo paso a paso
∫02 ∫01 (x + 2y) dy dx

Integral interna:

∫01 (x + 2y) dy = xy + y² |01 = x + 1

Integral externa:

∫02 (x + 1) dx = x²/2 + x |02 = 4

Resultado: 4 unidades acumuladas.

En Ciencia de Datos, una integral doble puede representar la acumulación de una métrica de riesgo sobre una zona geográfica o sobre una matriz continua de variables.

En Ingeniería Industrial, puede modelar la acumulación de temperatura, presión o defectos sobre una superficie de producción.

Integrales triples

Una integral triple acumula una magnitud en un sólido E. Si f(x,y,z) representa densidad, la integral triple calcula masa. Si representa concentración, calcula cantidad total de sustancia. Si representa temperatura acumulada, permite estimar carga térmica.

∫∫∫E f(x,y,z) dV
SistemaDiferencialUso recomendado
RectangularesdV = dx dy dzCajas, prismas, regiones con límites directos.
CilíndricasdV = r dz dr dθCilindros, tuberías, depósitos, simetría circular.
EsféricasdV = ρ² sinφ dρ dφ dθEsferas, domos, ondas, campos radiales.

Ejemplo académico

Calcular el volumen del sólido rectangular 0 ≤ x ≤ 2, 0 ≤ y ≤ 3, 0 ≤ z ≤ 4.

Desarrollo paso a paso
V = ∫02 ∫03 ∫04 1 dz dy dx

Primero: ∫04 1 dz = 4.

Luego: ∫03 4 dy = 12.

Finalmente: ∫02 12 dx = 24.

Resultado: el volumen es 24 unidades cúbicas.

En sistemas y software, una integral triple puede modelar carga térmica dentro de un centro de datos cuando la temperatura depende de la posición espacial. En multimedia, puede representar densidad de luz, partículas o color dentro de una escena volumétrica. En industrial, permite estimar masa, inventario físico o concentración dentro de un tanque.

Casos aplicados por carrera

Ingeniería Multimedia

Contexto: Una empresa desarrolla una escena 3D con niebla volumétrica. La densidad visual depende de la posición dentro del entorno.

Modelo sugerido: usar integral triple para estimar densidad total de partículas en un volumen de renderizado.

Interpretación esperada: relacionar el resultado con costo computacional, calidad visual y optimización de recursos.

Ingeniería Industrial

Contexto: Una planta mide concentración de calor en un tanque cilíndrico usado en un proceso productivo.

Modelo sugerido: usar integral triple en coordenadas cilíndricas.

Interpretación esperada: explicar impacto en seguridad, eficiencia energética y control de calidad.

Ciencia de Datos

Contexto: Una compañía analiza una función de riesgo distribuida sobre una zona de operación logística.

Modelo sugerido: usar integral doble sobre una región plana.

Interpretación esperada: asociar la acumulación con toma de decisiones y priorización de recursos.

Software y Sistemas

Contexto: Un robot de mantenimiento recorre una ruta dentro de un centro de datos; el esfuerzo depende de la dirección y magnitud de un campo de resistencia.

Modelo sugerido: usar integral en línea de campo vectorial.

Interpretación esperada: relacionar el resultado con consumo energético, mantenimiento predictivo y planeación de rutas.
